git log查看提交记录

git log 查看提交记录，参数：

-n (n是一个正整数)，查看最近n次的提交信息

git blame

如果你要查看文件的每个部分是谁修改的, 那么 git blame 就是不二选择. 只要运行’git blame [filename]’,你就会得到整个文件的每一行的详细修改信息:包括SHA串,日期和作者.
